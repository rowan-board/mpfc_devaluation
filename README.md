# Goal-directed weighting of outcome values in ventromedial prefrontal cortex 
Repository for: Board, Holton, Rushworth, & Manohar (in prep)

# Project overview
This repository contains the behavioural data, task code, and analysis code for a study investigating how goals/motivation influences choices in a novel decision making task (The two currecny task). The project explores the neural representations of goal directed action using fMRI, comuptational modelling and machine learning.

# Repository structure
- /analyse_timeCourses: contains matlab code and data to plot how neural activity (BOLD signal) evolves in the seconds following task events (e.g following the receipt of reward)
- /Devaluation_analyses: contains R code and behavioural data to analyse and plot the main behavioural effects from the task as well as the computational models used to analyse the data algorithmically
- /fmri_analyses: contains the main analysis scripts to conduct the activation based analyses, multi voxel pattern analysis, and connectivity analyses
- /task contains: the main script to run the behavioural task
